## Objective
Predict future sales trends using historical sales data.

## Features
- Data Preprocessing
- Sales Trend Analysis
- Linear Regression Model
- Future Revenue Prediction
- Visualization of Forecast Results

## Technologies Used
- Python
- Pandas
- Matplotlib
- Scikit-learn

## Dataset
Online Sales Data Dataset

## Output
Sales forecast graph displaying historical and predicted revenue trends.
